Organization by @metagn for general purpose Nim libraries deemed good enough for other people to use. The idea is that I should not be a chokepoint and other people can take over if development stalls on a project.

Currently a single library exists, will put up a table when more are added: https://github.com/holo-nim/holo-json

Candidate libraries: https://github.com/orgs/holo-nim/discussions/1
